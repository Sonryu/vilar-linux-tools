# vilar-linux-tools
🇧🇷:
Uma coleção de utilitários de linha de comando (CLI) e ferramentas de automação desenvolvidas em C++ para sistemas Linux. Este repositório foca em otimizar o fluxo de trabalho no terminal, automatizando tarefas repetitivas como gerenciamento de repositórios Git, manutenção de pacotes (DNF/APT) e monitoramento de sistema, utilizando a performance e a integração nativa que o C/C++ oferece com o kernel Linux.

🇬🇧:
A collection of command-line utilities (CLI) and automation tools developed in C++ for Linux systems. This repository focuses on streamlining terminal workflows by automating repetitive tasks such as Git repository management, package maintenance (DNF/APT), and system monitoring, leveraging C/C++ performance and its native integration with the Linux kernel.
